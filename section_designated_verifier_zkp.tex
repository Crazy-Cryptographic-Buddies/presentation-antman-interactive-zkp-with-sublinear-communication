\begin{frame}{Designated-Verifier ZKP}
	A ZKP is an interactive game between and prover $\Prover$ and $\Verifier$.\pause
	\begin{itemize}
		\item \textbf{Completeness:} $\Prover$ is honest then $\Verifier$ always trusts.\pause
		\item \textbf{Soundness:} $\Prover$ is unable to cheat.\pause
		\item \textbf{Zero-Knowledge:} $\Verifier$ achieves nothing beyond proof's validity.\pause
	\end{itemize}
	
	\textbf{Designated-Verifier ZKP (DVZK):} Verifier has secret verification key.
\end{frame}
